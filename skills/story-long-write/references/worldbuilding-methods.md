# 世界观与升级体系设计（番茄官方教程）

> 开新书搭设定、或写世界观 / 力量体系相关设定文件时加载。世界观 = 故事运转的底层框架，含**基础背景设定**（这世界什么样）+ **升级体系**（凭什么变强、怎么变强、代价是什么）。基础设定不是堆给读者看的，是让作者自己清楚主角活在什么世界里，故事才不会越写越乱。

---

## 一、基础背景设定

### 1. 要写哪些（挑最影响主线的写，不必全写）

1. **时空背景**：时代、空间、真实度。长篇不怕舞台大，但开始一定要有一个**主舞台**，别摊太散。
2. **社会结构**（决定"人怎么被分层"）：权力结构（谁掌权：皇权/宗门/财阀）、阶层结构（贵族vs平民、顶流vs糊咖）、身份规则（出身是否决定命运、有无阶层流动）。没有社会结构，人物关系会飘。
3. **运行规则**（最关键）：生存规则（靠什么谋生）、竞争规则（晋级/成名/变强靠什么）、交易规则（资源怎么流动）、惩罚规则（违规代价多大）。写清最影响主线的那几条。
4. **核心矛盾**：世界观靠的不是设定多，而是矛盾够清楚——阶层矛盾（贵族vs寒门）、权力矛盾（皇权vs世家）、资源矛盾（灵脉/名额/流量争夺）、价值矛盾（家族利益vs个人自由）。长篇一定要有"大矛盾背景"，后面剧情才能持续升级。
5. **超凡/特殊系统**（玄幻/科幻等必填）：力量来源、力量等级与差距、使用代价与获得方式、对社会秩序的影响。核心原则：力量体系要让读者觉得它**稳定存在**，不是作者现编的。
6. **主角的入场方式**：主角出身在世界哪一层？最先撞上的规则是什么？后期升级能否自然接入更大的世界？

### 2. 撰写思路（先立核心，再扩外层）

1. 定题材和主舞台（只定主角最常接触的那层世界）
2. 定最重要的 3 条规则（只写最影响主角命运的）
3. 定 2-3 组核心对立关系（嫡vs庶 / 正道vs魔道 / 老牌资本vs新人）
4. 定主角入场方式（弱者求生 / 重生复仇 / 底层逆袭）
5. 先立核心，等主线跑起来再逐步扩充外层地图、更高层势力、历史旧案——**别开头全是伏笔**。

### 3. 世界观框架模板（填空）

| 框架 | 填什么 |
|---|---|
| 世界类型 | 古代/现代/架空/玄幻/科幻 |
| 一句话概括 | 这是一个 xxx 的世界（例：强者为尊的玄幻世界，强者掌控命运） |
| 时空背景 & 当前局势 | 时代（什么时代）/ 当前状态（安稳/动荡/繁华/衰败/新旧交替）/ 当前最大问题（眼下最突出的局势和压力） |
| 世界运转逻辑（2-4个） | 权力/金钱/身份/家族/名声/武力/信息… 挑主导的几条 |
| 人群分层 | 上层 / 中层 / 底层 / 边缘但关键的人——写出不同身份的位置差和距离 |
| 普通人的生存方式 | 靠什么活着 / 最怕失去什么 / 改变命运的路径——让世界观落到日常生计 |
| 核心规则 | 规则1/2/3：这个世界默认遵守什么 |
| 核心矛盾（2-3组） | 个人vs家族 / 理想vs生存 / 旧秩序vs新变化 / 名声vs利益… |
| 关键势力及关系 | 能影响世界格局走向的力量体 |
| 主舞台 | 地点 / 表面上（是什么）/ 实际上（藏着什么）——把世界观落到一个具体场景 |
| 特殊系统 | 力量来源 / 等级划分 / 代价限制——力量从哪来、怎么变强、变强付什么代价（没有就填"无"） |
| 主角与世界 | 主角身份 / 主角目标 / 世界如何阻碍主角 / 世界如何逼主角成长 |
| 故事推进靠什么 | 身份变化 / 关系变化 / 资源争夺 / 事件牵连 / 局势骤变… |

> 番茄给了三个填好的范例：通用示例、传统玄幻（万法末世·灵脉枯竭·大罗剑宗vs九幽冥殿·主角荒古圣体）、豪门总裁（现代海城·顾叶林唐四大家族·契约未婚妻）。照结构填即可。

## 二、升级体系设计（有成长体系的题材：玄幻/修仙/都市高武/玄幻言情等）

核心不是"怎么升更爽"，而是**"为什么能升、为什么不能乱升、升了会带来什么代价"**。把升级体系当成一套带成本、带门槛、带副作用的成长规则系统来设计，后期才不容易崩。

### 1. 六步设计

1. **先定底层能源【重点】**：角色凭什么变强？只选 1 主 + 1 次（主：灵气/血脉/科技/信仰/法则领悟；次：悟性/天赋/心境）。别叠太多来源（灵气+血脉+神器+气运），前期丰富后期必失衡。
2. **控制阶梯数量【重点】**：大阶 **4-7 个**最好把握（每大阶再分前中后期）。大阶=质变，小层=量变。5 大境界×3 小阶=15 台阶，比"1到100"更能让每次突破都是剧情高潮，且不易战力崩坏。
3. **定每阶质变内容【重点】**：每升一大阶必须有清晰的**质变理由**——这一阶比上一阶本质变了什么。前期身体/能量积累 → 中期技巧/结构成形 → 后期法则/领域/世界级权限。（炼体：徒手碎石近战 → 聚气：真气外放剑气中远程 → 御空：凌空飞行、神识扫描数里）
4. **让升级与世界结构绑定**：个人变强路径要和社会资源分配一致。回答三问：①高阶资源被谁掌握？②普通人能升到哪步？③升级后社会地位是否改变？让升级和现实利益挂钩（"我要活下去""我要被尊重"），读者才在乎他能不能升上去。
5. **提前设计"例外机制"**：主角可以特殊，但必须写清来历与边界——①例外来源（双核心/极罕见血脉）②例外优势 ③例外代价（消耗寿元/因果）④例外边界（只在前三阶有效）。建议①②必填，③④至少写一个，防主角光环失控、破坏世界观平衡。
6. **给每阶补"输入—过程—输出"**：升级有前提、有过程、有结果。（输入：灵石100+完整功法+一次高压战斗；过程：压缩能量、冲击经脉、稳定核心；输出：技能+1、寿命+20、感知+50）

### 2. 六个常见问题 → 解决办法

| 常见问题 | 解决办法 |
|---|---|
| 资源设定前后不一，削弱世界观 | 关键资源先定三件事：稀缺度/获取渠道/使用门槛。低阶普及、高阶稀缺；资源由珍变常见必须补原因（新产地/技术改良/主角进了更高圈子）。定期回看前文查资源规则有没有被剧情临时改掉 |
| 只有等级、没有机制，无升级实感 | 先明确凭什么变强（来源清晰）+ 设"承载上限"（经脉承压/精神力阈值/肉体极限）；每升一阶对应一种明确变化（体质增强/感知扩大/技能上限提升），不只写"更强了" |
| 例外太多，常规体系成摆设 | 常规是底层基础，例外只制造冲突与亮点、不可反客为主。每种特殊都回答：来源是什么、代价是什么、能否复制（受天地排斥越到后期越难复制） |
| 金手指过于万能，挤压其他角色 | 先定核心方向再主动保留短板（能看穿谎言但身体极差、单独无战力）；再借世界规则限制泛用（瞬学任何技能但每次消耗等量寿命，复制绝学折寿十年）。一项能力别同时攻/疗/防/控全能 |
| 越阶战常态化，等级压制失去意义 | 越阶取胜=特殊事件，条件极苛刻且提前铺垫（反派被环境克制/主角付巨大代价）。叙事上从源头压低越阶战频率，必要时再加规则硬限制 |
| 能力没副作用，后期失控 | 越强的能力越要附成本（体力/精神/寿命/冷却/因果反噬/社会后果）。副作用不必每次剧烈，但要长期稳定存在，让角色在"要不要动用"间真实权衡 |

### 3. 升级体系模板（每阶都填）

| 模块 | 填什么 |
|---|---|
| 底层规则 | 力量来源 / 力量如何运行（依托什么、决定效率容量操控）/ 天赋·资源·悟性各影响什么 |
| 阶梯拆解（每阶都写） | 阶位名称 / 本阶核心质变 / 新增的能力或资格 / 代表性技能或表现 |
| 突破规则 | 必要资源 / 必要前置条件 / 触发方式（闭关/战斗危机/秘境感悟/仪式）/ 失败后果（气海受损/境界倒退/灵脉断裂/陨落） |
| 限制条件 | 资源限制 / 承载限制 / 规则限制（不同区域法则强弱）/ 社会限制（功法垄断、出身师承）/ 主角例外说明与代价 |
| 世界联动 | 哪些势力掌握本阶资源 / 本阶在社会中的地位 / 本阶是否受法律或组织约束 / 本阶会引发哪些新的敌人或责任 |
