# 写法自查（中国网文通用 · 番茄轻小说流）

> **定位**：中国网络小说（爽文向）通用写法自查清单。一～三是铁律（违反即重写），四～十是分项自查（写前对照，写完逐条过）。与某一本书强绑定的红线（角色信息边界、专有名词表、伏笔清单）不放这里，放该书 CLAUDE.md 与 设定／追踪。换书或换平台时，只需重审「二、排版铁律」，其余通用。
>
> **读者画像**：手机上快速下滑的普通人。句子一绕、理解成本一高，就直接划走。任何一段正文先过「小白文」这一关，再谈文采——写得再美，读者读不下去，就是失败。

**总纲：白描打底，话演推动（每章话演率不低于 60%，台词量按章型换挡——见九），短句直给，章章有钩，节奏张弛。**

## 一、小白文铁律（最高优先级，违反即重写）

1. **一句话只讲一个意思**。能拆两句就拆两句。禁止从句套从句、一段塞一堆信息。同一个意思只说一遍，不换着说法复读。
2. **短句为主**。长句一律拆开，主谓宾清楚，读一遍就懂，不用回头重读。
3. **少想、多做**。压缩内心分析，多写他做了什么、说了什么。「聪明」要靠一个具体动作或一句戳心窝的话直接砸出来，让读者一眼看懂他牛在哪，而不是让他在心里长篇盘算。
4. **爽点直给、当场见效**。反制／打脸那一下要干脆、明显、解气：前面把危机铺足，后面对手当场破防（变脸、语塞、失态），读者一看就爽。
5. **不装深沉、不文艺腔、不掉书袋**。比喻要么生活化、要么不用。网文的「好」是**带劲**不是**优美**：别写读者想划线摘抄的文艺金句（「像浸了毒的琉璃」「值一整条命」一律废掉）；主角要的是短而狠的**人物金句**（狠话、场面话、定场词），靠内容硬，不靠修辞美。
6. **理解成本压到最低**。宁可直白，不要高级。
7. **心理外化但要短**。用一个身体反应带过（后背冒汗、手一紧、口水不敢咽），不写连篇意识流长句。

### 反面对照（真实踩坑示例）

- 太绕：`不是不怕，是他这具身体记不住怎么怕才对，他这颗脑子却比谁都清楚，在死亡贴脸的时候乱动等于自杀。`
  → 改：`身体在怕。脑子却很清楚：这时候乱动，就是找死。`
- 太文：`他把这瞬间的悸动压回心底，像压下一份还没读完的报告。`
  → 改：`他把这点心思压下去。能不能活过今晚都难说，没工夫琢磨它。`
- 主角光在脑子里分析、内心太长 → 拆成短句，让动作和台词替他说话。

## 二、轻小说排版铁律（番茄阅读器）

对标已发布番茄轻小说的排版，正文采用「轻小说形式」：

1. **短段为主**：一句话／一个信息点就独立成段。冲击句、悬念句、对话，全部各自成段。
2. **段落之间空一行**（轻小说／番茄标准排版）。本条**覆盖**通用规则里「禁止段落间空行」——正文段落间用空行分段。
3. **对话独立成段**：用极短动作或提示引出，不用「他说道／她叹道」这类长标签。
4. **语言极度口语、直白、有网感**：该吐槽就吐槽，该大白话就大白话。
5. **节奏飞快，悬念层层递进**：每一小段抛一个小钩子，一句一句把读者往下拽。
6. **可适度用 …… 和 ——**：做停顿、拟声、悬念、话没说完（如「轰隆——」「我……是谁？」）。服务节奏，别滥用注水。
7. **长短交错**：冲击／悬念一句一段，叙述／推理可两三句一段，别通篇一句一段像提纲。
8. **基调跟着剧情走**：高压死局照样写高压。「轻小说」只指排版和语言节奏，不是要求内容搞笑。

> 与旧规则冲突时（.claude/rules/story-format.md 的「禁止空行 / 正文无 ……／——」），本项目一律以上方「小白文铁律／轻小说排版铁律」为准。

## 三、话演推动铁律（剧情主要靠「话」和「演」往前推）

**核心：小说主要靠「话」（对话）和「演」（表演/动作）往前推，不靠旁白解释。对话天生最像真人、最难被判 AI——情报、性格、张力全从嘴里出，你一句我一句地斗、试探、吓唬，别写成主角躺着看半天的独角戏。**

1. **能用对话推的，别用旁白讲**：信息、冲突、关系变化，尽量让角色「说出来／做出来」，而不是作者在旁边「叙述出来」。
2. **对白有来有回**（示例见五、2）：接话、打断、反问、否认、顺势嘲讽、岔开——每句都在接上一句的情绪／信息，不是轮流念台词。
3. **对白承担功能**（示例见五、3）：每段关键对话至少干一件事——抛新信息／造冲突／做决定／改关系／推行动；说完局面要变一点。
4. **潜台词**（示例见五、4）：越重要的情绪越别第一句说破，藏进反话、停顿、答非所问里。
5. **少想多做**（同一、3）：主角的「聪明／情绪／判断」优先用一个动作、一句台词、对方一个反应砸出来；砍掉「他心里那本账」「他知道…」「他这么聪明」这类**自我解说式旁白**——读者已经从话和演里看到了，不用再讲一遍。
6. **旁白只留必要的**：藏身份／藏穿越这类读者必须知道、又不能让角色说破的内心，保留最短的一句；其余能外化就外化。
7. **禁止旁白过度说明（话和演演出来了，就别再贴一句标签解释）**：删掉一切「替读者理解」的旁白——
   - 标签化定性：`是个忠心的`／`她是真心疼他` → 披衣＋「小心着凉」已经演出来了，别再点一句。
   - 解释语气：`话里裹着刺`／`语气里全是恨` → 刺和恨藏在台词内容里，不点破。
   - 点破表情含义：`那点意思明明白白写在脸上：看又露馅了` → 只写「他挑了挑眉」，剩下留白给读者。
   - **替角色总结感受**：`她头一回觉得比放血还难熬` → 靠她脸红、尾巴缩这些「演」来传。
   - 碎旁白填充：`沈砚没推。`／`她在等他露怯。` 这类只交代态度的短旁白，能并进动作/台词就并、能删就删。
   > 一句话：**能演出来的，绝不旁白解释；旁白只留角色此刻真切感知到、且非交代不可的（如藏身份）。**

> 例外：开局黄金三章的「机制刚需」内心戏（藏身份、藏穿越、金手指独自觉醒这类非交代不可的博弈）可放宽；**从第 4 章起，卡 BUG／推理／机制揭示也必须靠话演推动**，不得写成大段内心独白＋旁白解释。

## 四、章节工程：钩子·爽点·期待感（追读的命）

1. **章尾必卡钩**：每章最后一段必须落在钩上——危机钩（刀架到脖子上）、反转钩（真相掀一角）、期待钩（下章要打脸／要兑现）、悬念钩（新谜面砸出）。禁止圆满收尾、平静收尾。
2. **章头接钩**：开章三段内接住上一章的钩，别先写景、倒叙、切别的线泄气。
3. **每章必有所得**：读者读完这章至少拿到一样——一个爽点、一层新信息、一步关系推进。三样全无＝水章，回炉。
4. **压抑有上限**：主角憋屈／被压制不能连着超过一章，必须给一次小回击或小反制；大打脸可以攒着，小出气不能断。先抑后扬，抑不过量。
5. **期待感三步**：立目标（读者知道主角要什么／要收拾谁）→ 途中亮牌或受阻（读者替他急）→ 兑现超预期（读者爽到）。兑现别拖过铺垫后三章，拖久了期待馊掉。
6. **行动可跟随**：主角每步棋，读者要么当场看懂图什么，要么一章内看到它见效。藏底牌可以，藏到读者看不懂主角在干嘛不行。
7. **关键设定隔章重提**：读者记性差，核心规则／在场伏笔每隔 3-5 章借剧情自然带一次。重提≠复读——换个角度、换张嘴、换个场景说。
8. **章节名当钩子**：带信息量＋悬念或爽点预告（谁＋干什么＋反差），不起文艺空名。
9. **开新书黄金三章自查**：主角、金手指、核心冲突（死局／目标）三件套前三章内全部立住；第一章内必须出冲突；金手指要带代价和限制才立得住；背景设定拆碎了随剧情喂，禁止开局堆世界观。

## 五、对白手术（示例库，配合「三」使用）

1. **一开口见人设**：身份、年龄、脾气、文化水平直接决定措辞和语气——人物一张口，读者就知道「这个人是谁」。配角同理：哪怕工具人，开口三句内也要给一个记忆点（口癖／立场／毛病），不许所有人一个腔调。

```
改前：
父亲说：“我希望你以后能够更加重视自己的身体健康，毕竟昨天晚上你工作到太晚了，这种行为是不正确的。”
儿子说：“好的，我知道了，我以后会注意休息，也谢谢你的关心。”

改后：
父亲皱着眉，把桌上的豆浆往他面前一推：“一宿没睡？”
儿子低头拿包子：“睡了俩小时。”
“你那也叫睡？”父亲冷哼一声，“再这么熬，下回别说头晕，抬不起来都活该。”
儿子嘴硬：“哪有那么夸张。”
父亲瞪他一眼，声音却压低了点：“少废话，先把早饭吃完。”
```

> 改后父亲的话带训带硬，儿子嘴硬但领情。关系感自然流出来，不需要再解释「父亲很关心儿子」。

2. **有来有回**：互动感的核心是「回应」，不是「轮流发言」。接话、打断、反问、抢话、吐槽，都算回应。

```
改前：
林舟说：“你今天怎么来得这么晚？”
周凯说：“因为路上堵车了。”
林舟说：“原来如此，那我们现在进去吧。”
周凯说：“好的，我们进去。”

改后：
林舟看了眼时间：“你再晚五分钟，我都准备给你立碑了。”
周凯把车钥匙一甩：“少来，堵了一路，我能活着到这儿已经算命大。”
林舟挑眉：“你上次迟到也这么说。”
周凯面不改色：“说明这理由稳定，值得长期使用。”
林舟被气笑了，抬腿往里走：“滚，奶茶你请。”
周凯跟上去：“行，但你别点最贵的，我今天是受害者。”
```

3. **承担功能**：每段关键对白至少干一件事——给信息、造冲突、做决定、改关系、推行动。说完局面要变，哪怕只是态度变了。

```
改前：
陆沉冷笑：“你果然还是来了。”
裴烬看着他：“我当然会来。”
陆沉说：“来了也没用，你赢不了我。”
裴烬神色冰冷：“那可未必。”
陆沉眯起眼：“你可以试试。”

改后：
陆沉立在祭坛高处，垂眼看他：“你来晚了，镇魂印已经开了两层。”
裴烬脚步一顿，目光猛地落向他身后的血阵：“你拿百姓祭阵？”
陆沉笑了一声：“不然你以为，城南那一百三十七口人，是怎么一夜失踪的？”
裴烬手指骤然收紧，声音沉了下去：“他们果然在你手里。”
“现在才猜到？”陆沉抬手一挥，石门轰然落下，“可惜，退路没了。要么杀了我，毁阵；要么再拖半刻钟，等着给他们收尸。”
裴烬盯着他，缓缓拔剑：“那你今天就别想活着走出去。”
```

4. **潜台词**：嘴上抱怨实际关心，嘴上无所谓实际试探。越重要的情绪，越藏进反话、停顿、答非所问、刻意平静里——让读者参与理解，黏性才高。

```
改前：
女生说：“我现在很生气，因为你昨天没有回复我的消息，而且你和别的女生一起吃饭，我吃醋了。”
男生说：“对不起，我知道你吃醋了，我以后不会这样了。”

改后：
她低头搅着杯子里的冰块，语气很淡：“昨天挺忙啊。”
他一顿：“手机没电了，后来应酬也没顾上看。”
“哦。”她点点头，“我还以为，你跟那位一起吃饭，开心的忘了我呢。”
他听出不对，往前坐了点：“你看见了？”
她终于抬眼看他：“看见倒没什么，主要是某些人，连解释都省了。”
他沉默两秒，低声说：“是我不对。”
```

## 六、句子手术（示例库，配合「一」使用）

1. **长句拆开**：一句话好不好，先看通顺、清晰、有没有画面。

```
改前：
他推开门之后看到客厅里一片狼藉地上到处都是碎玻璃而母亲坐在沙发边低着头一动不动他心里一下就慌了。

改后：
他推开门，客厅一片狼藉，地上全是碎玻璃。母亲坐在沙发边，低着头，一动不动。那一瞬间，他心里猛地一慌。
```

2. **信息拆层**：挤成一团的内容拆成主干句＋补充句。先写最重要的动作，再补心理、原因、状态——轻重立住，重点才立住。

```
改前：
她在看见走廊尽头那扇本该锁着的门竟然半掩着之后，呼吸一下就停住了，站在原地不敢继续往前走，脑子里飞快闪过昨晚听到的那些传言，越想越觉得后背发凉，连手心都渗出了汗。

改后：
她猛地停下脚步，盯住了走廊尽头那扇门。门竟然是半掩着的。她一时没敢再往前走。昨晚听来的那些传言，忽然全涌了上来。她越想越冷，连手心都慢慢渗出了汗。
```

3. **说明改画面**：别急着告诉读者他是什么情绪，让读者从动作里看出来。适用于情绪、性格、心理活动——凡是想说「他是个什么样的人／他现在什么感受」，都优先动作化。

```
改前：
他现在非常生气，因为他觉得自己被羞辱了，所以整个人的情绪都很不好。

改后：
他下颌绷紧，手背上的青筋一点点鼓了起来。那句讽刺还在耳边打转，他紧紧盯着对方，一个字都没回。
```

4. **抽象改具体**：「很破」「很冷」「很吓人」读者想象不出来，拆成能感知的细节。短不等于单薄，短而准才有力量。

```
改前：
她在听完这句话之后，内心当中立刻就产生了一种非常明显的慌乱情绪，整个人一下子不知道该怎么办才好。

改后：
她一听这话，心口猛地一乱，站在原地都忘了接下一句。
```

5. **通用改独家**：别写放之四海皆可的描写，写这个场景、这一刻独有的细节。「药味很重」→ 写这间屋、这一刻它怎么冲、冲得人什么反应。

## 七、去 AI 味自查

1. **识 AI 味五特征**：刻板句式、情感空洞、过渡生硬、用词模式化、节奏匀速——见一处改一处。
2. **句式长短彻底错开**：有时一长串逗号怼到底，有时俩字一句。别匀速、别每段一样长。
3. **AI 习语与条理词清零**：首先／其次／最后／综上／由此可见／值得注意的是／不难发现／这意味着／也就是说／换句话说——一律不用，改成顺着思路的自然衔接。
4. **排比对仗清零**：排比、对仗工整句是 AI 重灾区（「鳞是冷的，肉是热的」这种），坚决不写。
5. **高频小动作轮换**：挑眉／眯眼／勾唇／瞳孔一缩这类身体反应词，同一个一章内不超过 2 次——换着写，或者干脆删。
6. **章末禁总结升华**：「他终于明白了……」「这一夜注定无人入眠」这类收尾一律砍掉。章末归钩子（见四、1），不归感慨。

## 八、视角与信息纪律

1. **单场景单视角**：一场戏钉死跟一个人的眼睛走，禁止头顶摄像机来回跳。「电影式硬切镜头＋强行事后解释」在网文里造成极大阅读撕裂感，迫不得已才用，切完立刻锚定新视角。
2. **叙述带腔调**：叙述跟着视角人物的眼睛和偏见走——他嫌弃就写嫌弃，他觉得晦气就骂晦气。别当中立的旁白摄像机。
3. **台词前核知识边界**：开口前先问——这个角色知道这件事吗？懂这个词吗？不知道的，绝不能从他嘴里说出来；该角色不在场时得到的信息，他不能凭空知道。（本书的具体边界红线见 CLAUDE.md。）
4. **新名词先核后写**：新武器／新机构／新材料／新规则落笔前，先 grep 前文与设定确认建立过；没建立过，要么先补建立场景，要么改用已有的。凭空发明即打回。

## 九、节奏张弛与台词配比

**核心原则**：让读者「喘得过气」，但「停不下来」。

剧情不是一路爽到底，也不是持续高压虐到底，而是有张有弛、爽虐交替、燃暖并行：高压段落之后必有呼吸空间，让读者的情绪有落点也有蓄力。

- 高潮／战斗段：极短句加速，砍掉一切拖节奏的修饰。
- 日常／温情段：可以放缓，但必须埋点——下一波冲突的引线就藏在这段舒服日子里。

### 台词量按章型换挡

同一本书里，对话密度必须跟着章节类型换挡——日常靠说话养人，对质靠说话开战，决战靠动作压场：

| 章节类型 | 台词量 | 要点 |
|---|---|---|
| 日常过渡章 | 50%–70% | 靠角色互动、插科打诨推进日常，读起来不累；话多但每句带信息、带笑点或带关系变化，不许车轱辘闲聊（另见四、3）。 |
| 冲突爆发章（撕逼／对质） | 60%–80% | 对话就是战场：挑衅→回击→再加码，一句比一句狠，密集交锋把矛盾顶到最高点（拉爽点）；打脸的那句必须出自主角之口，不许旁白代劳。 |
| 高潮战斗章（打斗／大决战） | 10%–20% | 生死看淡，不服就干。高手过招忌废话，以动作、招式后果、局势变化为主；台词只留最狠的一两句（宣战、翻盘、定场词），一句顶一百句。 |

- **口径**：按「对话段落」（台词＋贴身引出动作）占全章篇幅估算；若只数引号内字数，各档相应下浮。不必卡死小数点，要的是「这一章该话密还是话疏」的方向感——写完通读一遍，日常章闷了就是话少了，战斗章拖了就是话多了。
- **混合章拆场景换挡**：情感／暧昧章——斗嘴拉扯段按冲突档跑，进入身体戏后按战斗档收话、改走动作与体感；查案／解密章——走访盘问按冲突档，勘验推演按战斗档。一章之内跟着场景切，不吃一档到底。
- **与话演率不冲突**：战斗章台词掉到一两成，但「演」（动作）顶上来，话＋演合计推动仍不低于 60%（总纲）——任何章型，话少了都只能拿动作补，不许拿旁白解说顶班。

**举例**：《我在精神病院学斩神》燃与暖精准交替，高强度战斗后插入姨妈的鸡蛋、小黑癞蹭脚踝等日常温情，让读者始终有情感着陆点。

**目的**：
1. 阅读黏性——一路高压读者疲惫弃书，一路爽读者麻木无感；张弛有度才能一直新鲜。
2. 情感共鸣——爽虐交替让每一次爽被放大：读者知道这个爽是拿命换来的，不是白给的。
